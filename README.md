# Curator

Yet again another rss reader with podcast support. 

# Features:

* All data is in local database
* Rss and Atom
* Read/unread
* Dark Mode
* Bookmark
* Zen Reader (Experimental)
* Podcast Support
* Hold to view the summary of rss feed item
* Categories
