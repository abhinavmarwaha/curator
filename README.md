# Features:

* All data is in local database
* Runs on mobile, no need of external server
* Rss, Atom and OPML support
* Read/unread
* Dark Mode
* Bookmark
* Zen Reader (Experimental)
* Podcast Support
* Hold to view the summary of rss feed item
* Categories
